# AI Guideline: Creating Engaging LinkedIn Carousels with Canva

## Overview

This guideline provides best practices for AI-assisted creation of high-performing LinkedIn carousel posts. Carousels achieve the highest engagement rate (6.60%) of all LinkedIn content formats, outperforming single images (4.85%), videos (5.60%), and text-only posts (~4%).

---

## 1. Technical Specifications

### Dimensions & Format
| Spec | Requirement |
|------|-------------|
| **Optimal Size** | 1080 × 1350 px (4:5 ratio) |
| **Alternative** | 1080 × 1080 px (1:1 square) |
| **File Format** | PDF only (LinkedIn converts to swipeable carousel) |
| **Max File Size** | 100 MB |
| **Max Slides** | 20 slides |

### Typography
| Element | Minimum Size |
|---------|--------------|
| Headlines | 24pt |
| Body Text | 18pt |
| Words per Slide | 25–50 words max |

## 2. Carousel Structure

### Optimal Length
- **Ideal**: 10–12 slides
- **Minimum**: 5 slides (fewer = 35% less reach)
- **Maximum**: 20 slides (15–20 = diminishing returns)

### Slide Framework

```
Slide 1: HOOK
├── Bold, attention-grabbing headline
├── Benefit-driven statement or provocative question
├── Visually striking design
└── Clear value proposition

Slides 2–11: CONTENT
├── One key point per slide
├── 25–50 words maximum
├── Visual hierarchy (headline → supporting text)
├── Consistent layout and branding
└── Progressive storytelling flow

Slide 12: CTA (Call-to-Action)
├── Clear next step
├── Engagement prompt (comment, follow, save)
├── Optional: Lead magnet or link reference
└── Logo/branding reinforcement
```

## 3. Content Types That Perform Best

### High-Engagement Formats
1. **Step-by-Step Guides** – Break down processes into actionable slides
2. **Listicles** – "7 Tips for...", "5 Mistakes to Avoid..."
3. **Before/After** – Transformation stories with visual proof
4. **Frameworks & Checklists** – Actionable takeaways users can implement
5. **Data Storytelling** – Statistics with visual interpretation
6. **Personal Stories + Lessons** – "How I..." or "Why I..." narratives
7. **Myth vs. Reality** – Contrasting common beliefs with facts
8. **Case Studies** – Real results with specific metrics

### Content Rules
- Lead with value, not promotion
- Each slide should stand alone but contribute to the narrative
- Include relatable examples or analogies
- Avoid jargon unless targeting a specific technical audience

## 4. Design Principles

### Visual Consistency
- **Color Palette**: Stick to 2–3 brand colors throughout
- **Fonts**: Use max 2 font families (1 headline, 1 body)
- **Layout**: Repeat similar structures for each slide type
- **No Company Naming**: Never mention "Simplefysed" or "Simplefysed Solutions" anywhere on carousel slides — not in footers, headers, watermarks, or body text. Company branding makes the post look like an ad, which LinkedIn's algorithm deprioritizes and audiences skip. Let the value speak for itself. The company profile already appears next to the post on LinkedIn.

### Mobile-First Design
- Test readability on small screens
- Keep text centered (avoid edges—profile pictures overlay corners)
- Use high-contrast color combinations
- Ensure tap targets are adequately spaced

### Space & Layout Usage — CRITICAL RULES

**Canvas Fill Requirements (Mandatory):**
- Content must occupy **at least 70%** of the total slide height (1350px = content should span ~945px minimum)
- **Safe zone padding**: 60px from all edges (content area = 960px × 1230px)
- Content should be **vertically distributed** across the canvas — not clustered in one zone with large empty gaps

**Vertical Content Distribution:**
- Divide the slide into 3 vertical zones: TOP (0-450px), MIDDLE (450-900px), BOTTOM (900-1350px)
- **At least 2 of 3 zones** must contain meaningful content on every slide
- If content only fills TOP zone, add: supporting visuals, icons, dividers, source citations, or pull-quotes to MIDDLE/BOTTOM zones

**Statistic/Number Slides (e.g., "40%", "90%", "$2M"):**
- Large stat: 150-200pt font, centered vertically in MIDDLE zone
- Supporting text below stat: 24-32pt, explaining what the number means
- Source/citation: 16-18pt at bottom of slide (e.g., "— Gartner, 2026")
- Optional: Add an icon, subtle graphic, or accent line to fill space
- **Never leave the bottom third empty** on stat slides

**Element Sizing to Fill Canvas:**
- Headlines: Scale up to 48-72pt if content is minimal
- Body text: Scale up to 24-32pt for better readability and canvas fill
- Icons/visuals: Minimum 120px × 120px, scale up to 200-300px if space allows
- Comparison boxes (Before/After): Minimum height of 250px each

**Two-Column Layouts:**
- Each column must be at least 420px wide (with 60px gutter)
- Text must remain readable at 18pt+ on phones under 5.5 inches
- If text doesn't fit at 18pt+, **stack vertically instead** of squeezing columns

**Common Layout Violations to Avoid:**
- ❌ All content in top half with bottom half empty
- ❌ Floating stat with tiny supporting text
- ❌ Comparison boxes that are too small for the canvas
- ❌ Text squeezed to fit rather than layout adjusted

### Accessibility
- Minimum contrast ratio: 4.5:1 for body text
- Avoid red/green color combinations
- Use clear, sans-serif fonts
- Add alt text when uploading to LinkedIn

### Visual Elements
- Use authentic photos over stock images when possible
- Include icons to reinforce key points
- Add arrows or visual cues to guide reading flow
- Use whitespace strategically—don't overcrowd slides

## 5. Slide-by-Slide Best Practices

### Slide 1: The Hook
**Goal**: Stop the scroll and promise value

**Elements**:
- Headline: 5–10 words max
- Subheadline: Clarify the benefit
- Visual: Eye-catching but not cluttered
- No logo needed (save space for impact)

**Hook Formulas**:
- "X Things I Wish I Knew About [Topic]"
- "Stop Doing [Common Mistake]"
- "The [Timeframe] Guide to [Outcome]"
- "[Number]% of [Audience] Get This Wrong"
- "What [Authority] Won't Tell You About [Topic]"

### Slides 2–11: The Content
**Goal**: Deliver value incrementally

**Structure per slide**:
```
[Number/Icon]
HEADLINE (4-8 words)
Supporting text (1-2 short sentences)
[Optional: Visual element]
```

**Tips**:
- Number your points for easy navigation
- Use parallel structure across slides
- Include a "pattern interrupt" slide mid-way (quote, statistic, or visual break)
- Vary layouts slightly to maintain interest

### Final Slide: The CTA
**Goal**: Convert attention into action

**CTA Types**:
- **Engagement**: "Comment your biggest takeaway"
- **Follow**: "Follow for more [topic] tips"
- **Save**: "Save this for later"
- **Lead Gen**: "DM me [keyword] for the full guide"
- **Link**: "Link in comments for [resource]"

**Design**:
- Make CTA visually distinct
- Include your name/handle
- Add profile photo for personal connection
- Reinforce branding

## 6. Caption & Posting Strategy

### Caption Structure
```
[Hook - 1-2 lines that relate to carousel topic]

[Brief context or why this matters - 2-3 lines]

[Call-to-action: Swipe to learn...]

[Hashtags: 3-5 relevant tags]
```

### Optimal Posting Times
- **Best Days**: Tuesday–Thursday
- **Best Times**: 9:00–11:00 AM (target audience's local time)
- **Frequency**: 1–2 carousels per week maximum

### Engagement Boosting
- Reply to every comment within first 2 hours
- Engage with others' posts 30 min before and after publishing
- Ask a question in the caption to spark discussion
- Tag relevant people sparingly (only if genuinely relevant)

## 7. Canva-Specific Workflow

### Template Setup
1. Create new design: Custom size 1080 × 1350 px
2. Set up brand kit (colors, fonts, logo)
3. Create master slide layouts:
   - Title slide template
   - Content slide template (2–3 variations)
   - CTA slide template

### Design Process
1. Outline all slide content in text first
2. Apply content to templates
3. Add visual elements (icons, images, shapes)
4. Review for consistency across all slides
5. Export as PDF (File → Download → PDF Standard)

### Canva Pro Features (if available)
- Brand Kit for consistent styling
- Magic Resize for multi-platform adaptation
- Background Remover for clean visuals
- Content Planner for scheduling

## 8. Quality Checklist

### Before Publishing
- [ ] Hook slide is compelling and benefit-driven
- [ ] Each slide has one clear point (25–50 words)
- [ ] Visual consistency across all slides
- [ ] Text is readable on mobile (18pt+ body, 24pt+ headlines)
- [ ] CTA is clear and actionable
- [ ] No company name ("Simplefysed") appears anywhere on slides
- [ ] **Canvas fill check**: Content spans at least 70% of slide height (2+ of 3 vertical zones)
- [ ] **Stat slides**: Include supporting context and source citation (not just the number)
- [ ] **No squeezed elements**: All text/boxes at proper size, not shrunk to fit
- [ ] No slide has more than ~30% unused empty space
- [ ] PDF exports correctly (check page order)
- [ ] Caption includes relevant hashtags (3–5)
- [ ] No spelling or grammar errors

### Performance Metrics to Track
- Impressions
- Engagement rate (likes + comments + shares / impressions)
- Dwell time (time spent viewing)
- Saves (high-intent signal)
- Profile visits
- Follower growth from post

## 9. Common Mistakes to Avoid

| Mistake | Why It Hurts | Fix |
|---------|--------------|-----|
| Too much text per slide | Overwhelms reader, reduces dwell time | Max 50 words, use visuals |
| Inconsistent design | Looks unprofessional | Use templates, brand kit |
| Weak hook | No one swipes past slide 1 | Lead with benefit or curiosity |
| No CTA | Wasted engagement opportunity | Always include clear next step |
| Overly promotional | LinkedIn penalizes promotional content | Lead with value, soft-sell only |
| Company name on slides | Looks like an ad, reduces organic reach | Never put "Simplefysed" on any slide — your profile is already visible |
| Large empty gaps | Slides look unfinished, waste mobile screen space | Distribute content across the full canvas, add visuals or context to fill |
| Content clustered in top half | Wastes 40-50% of mobile screen real estate | Use 3-zone rule: content must span at least 2 of 3 vertical zones |
| Tiny comparison boxes | Users can't read, looks amateurish | Comparison elements must be 250px+ height each |
| Stat slides with no context | Number floats without meaning | Always add source citation + supporting sentence below stat |
| Text squeezed to fit | Unreadable on mobile, looks cramped | Scale layout up, never shrink text below minimums |
| Poor mobile formatting | 60%+ users on mobile | Test on phone before posting |
| Using same format repeatedly | Algorithm reduces reach by ~30% | Vary content types |

## 10. AI Prompt Templates for Carousel Creation

### Content Generation
```
Create a 10-slide LinkedIn carousel outline about [TOPIC].
Target audience: [AUDIENCE]
Goal: [AWARENESS/LEADS/ENGAGEMENT]

For each slide, provide:
- Headline (5-8 words)
- Supporting text (1-2 sentences, max 40 words)
- Suggested visual element

Include a hook slide and CTA slide.
```

### Hook Generation
```
Generate 5 alternative hook headlines for a LinkedIn carousel about [TOPIC].
The target audience is [AUDIENCE].
Make them curiosity-driven, benefit-focused, or pattern-interrupting.
```

### CTA Optimization
```
Suggest 3 different CTA approaches for a LinkedIn carousel about [TOPIC]:
1. An engagement-focused CTA (drives comments)
2. A lead-generation CTA (captures DMs or emails)
3. A relationship-building CTA (grows followers)
```
